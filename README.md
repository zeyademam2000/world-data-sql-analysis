# 🌍 World Data SQL Analysis

## Project Overview
SQL analysis on the World Database exploring population, 
economics, geography and languages across all countries.

## 🗃 Database Tables
- country — population, GNP, surface area, life expectancy
- city — population by city
- countrylanguage — official languages per country

## 🔍 Questions Answered
1. Top 10 most populous cities
2. Continent with highest population density
3. Top 10 largest countries by surface area
4. Country with highest GNP per capita
5. Countries with English as official language
6. Average life expectancy per continent
7. Countries with highest population density
8. % of population living in capital city
9. Continents with most official languages
10. Average GNP per continent

## 🛠 Skills Used
SQL · PostgreSQL · JOINs · GROUP BY · CASE WHEN · 
ROUND · Aggregations · Type Casting
